# Exam Assistant

This project scaffolds an exam assistant application.

Current structure:
- `app/` contains the main application code, agents, tools, graph, ingestion logic, and prompts.
- `data/` contains `raw/`, `processed/`, and `vectorstore/` subdirectories for managing datasets.
- `tests/` is reserved for test modules.
- `requirements.txt` will list Python dependencies for the project.

Implementation code is currently placeholder-only.
